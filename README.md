# SolTravel

Static travel website for Morocco ↔ Nordics flight + hotel packages.

## Publish to GitHub Pages

1. Create a GitHub repository, for example `SolTravel`.
2. Add a remote to this local repo:
   ```bash
git remote add origin https://github.com/<your-username>/<repo-name>.git
```
3. Push to GitHub:
   ```bash
git push -u origin main
```
4. On GitHub, open **Settings > Pages** and select the `main` branch, `/ (root)` folder.
5. Visit the published URL: `https://<your-username>.github.io/<repo-name>/`

## Notes

- All site files are in `index.html`, `styles.css`, `script.js`, and the image files.
- The site is a static HTML/CSS/JS landing page, so GitHub Pages is a good hosting option.
